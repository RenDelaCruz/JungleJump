# Jungle Jump

A platformer made with the Phaser 3 engine!

![Jungle Jump Sample](assets/sample.gif)

## Startup

Clone this repository, then `cd` into the directory.\
Start the server with:

```python
python -m server
```

Open the browser at that port and start playing!
